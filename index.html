About me

As a Computer Science and Engineering undergraduate at the University of Moratuwa, Sri Lanka, I have acquired skills in Software Development using Java, Python etc.

Further I have excelled in Front-end development as well as Back-end development. Currently I have been on Second Class Upper in every semester. While I was following the Engineering course at university I have also focused on other aspects of studies to improve the diversity of my skill set. I have completed following professional qualifications.


• Diploma in Management Accounting (CIMA) (UK)

• Certificate Course in Human Resource Management (CIPM)

Further I’m an active member of the Rotaract Club of my university and a student member of IEEE (Institute of Electronic and Electrical Engineers) and IESL (Institution of Engineers Sri Lanka). I have contested in algorithm contests organized by IEEE as well.

I'm a passionate programmer and creative UX designer who is skilled in critical thinking, problem solving, analytical skills and leadership skills.

Services

Software Development

Designing,developing and maintaining both desktop and mobile applications for database management systems, File management systems, e-commerce applications, etc. using Java, C++, Python and other object oriented languages which are efficient and satisfying user requirements.

Front-end Development

Producing HTML, CSS and JavaScript for a website or Web Application so that a user can see and interact with them directly giving user an amazing experience with interesting user interfaces and required details with a blend of efficiency and creativity.

Back-end Development

Responsible for and focus on the inner workings of web applications to build out the "server-side" of web applications. Server-side is the code and language that runs behind the scenes on the web server or the back-end using languages like PHP plus database management.

4-bit Microprocessor (2017)

A simple processor which is capable of executing simple set of instruction

This project is done under the module Computer Architecture. A simple 4-bit microprocessor was built using BASYS2 micro-controller and Xilinx ISE. Here the registers, ROM and other components were made from the simple logic gates to get the architectural design of those components.

Lake Nozama (2017)

A simple simulated game done using Java.

This was one of my first projects using Java. The Object Oriented Principles (OOP) are used in programming this project. Multiple threads are implemented in this small game to get familiar with concurrent programming.
